# Drikk Opp - Drikkelek

## Formålet med appen
- Legge til rette for at flere drikkeleker skal befinne seg på samme plass.
- Drikkeleker som skaper stemning på vors både for de som liker et rolig vors, og de som er litt mer fest glade.


## Oppstart av appen
1. Installer nødvendige pakker: `npm install`.
2. Start expo: `npx expo start`
- Expo må muligens oppdateres. (Se fremgangsmåte for dette under)


## Oppdatere expo versjon
1. Oppgradere Expo SDK: `npx expo install expo@latest`

2. Oppgrader avhengigheter: `npx expo install --fix`

3. Kjør appen: `npx expo start`
